Subject Expertise/Taught

Cyber Security

Cybersecurity is the practice of protecting computers, servers, mobile devices, electronic systems, networks, and data from digital attacks, theft, and damage.

Information Security

Information security is the practice of protecting information and information systems from unauthorized access or threats. It involves implementing measures to ensure the confidentiality, integrity, and availability of information.

Data Structure

A data structure is a way of organizing and storing data in a computer so that it can be accessed and modified efficiently. Different types of data structures are suited to different kinds of applications, and some are highly specialized to specific tasks.

Database Management System

A database management system (DBMS) is a software application that interacts with the user, other applications, and the database itself to capture and analyze the data.

Programming in C

C is a high-level programming language that was developed in the 1970s. It is a widely used, general-purpose programming language that is known for its efficiency and flexibility. C is often used to develop operating systems, compilers, and other software that needs to be highly efficient.

Object Oriented Programming with C++

C++ is an object-oriented programming (OOP) language that was developed in the early 1980s. It is an extension of the C programming language, and it was designed to be an efficient and flexible language for developing a wide range of applications.

Computer Organization

Computer organization refers to the way that the hardware and software components of a computer system are arranged and interconnected. It involves the design of the hardware and software components of a computer, as well as the way that these components work together to perform tasks.
